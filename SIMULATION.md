## Parameter Calibration

### σr = 1.0% Justification

The rate volatility parameter was reduced from 1.5% (v0.3.3) to 1.0% in v0.3.4 following a 3×3 parameter sweep. The 2022 realized daily rate volatility in 10-year Treasury yields was approximately 5–8 bps/day annualized, corresponding to σr ≈ 0.8–1.2%. A value of 1.0% sits in the center of this range and produces stress path incidence of 22.9% — consistent with the target of 20–25% stressed scenarios for meaningful policy surface validation.

### ρ = 0.35 Justification

A sensitivity analysis across ρ ∈ {0.35, 0.45, 0.55} confirmed that marginal stress incidence and drawdown distributions are insensitive to the rate-spread correlation assumption at σr = 1.0%. Only emergency activation rates exhibited a modest monotonic increase with ρ (0.4% to 0.6%). The lower value of 0.35 was selected to avoid artificially forcing correlated tail events while preserving realistic joint stress dynamics.

## Output Files

### `summary.csv`
1,000-path simulation results. Key columns:

| Column | Description |
|---|---|
| `min_liq` | Minimum liquidity index across the path |
| `max_drawdown` | Peak NAV drawdown |
| `max_daily_liq` | Peak daily liquidation count |
| `emergency_days` | Days in emergency unwind state |
| `max_oas` | Peak OAS reached |
| `oracle_stale` | Boolean: path experienced oracle staleness |
| `defensive_activated` | Boolean: defensive state triggered |

### `sweep_results.csv`
Full 3×3 sweep results across σr ∈ {0.8%, 1.0%, 1.5%} × ρ ∈ {0.35, 0.45, 0.55}.

## Figure Generation

All figures (MC_Fig6 through MC_Fig10) are generated by `generate_figures.py`. Statistics are read directly from `summary.csv` — nothing is hardcoded.

```bash
python simulation/generate_figures.py
```

Figures are written to the directory specified in the script (default: current directory).

---

## Reproducibility

With `seed=42` and the parameters above, the simulation is fully deterministic. Running `simulation_v034.ipynb` end-to-end will reproduce `summary.csv` exactly and regenerate all Monte Carlo figures.
